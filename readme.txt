Easily add site-wide tracking codes and conversion pixels. Additionally manage all your social media links, and display them on your site using shortcodes.


== DESCRIPTION ==

**Simple Tracking** adds an admin interface where you can easily add your tracking and conversion pixel, such as Google Analytics, Facebook Pixel, and more.

Additionally manage all your social media links, and display them on your site using shortcodes.

= SUPPORTED SERVICES =

* Google Analytics
* Google Tag Manager
* LinkedIn Pixel
* Facebook Pixel
* Pinterest Pixel


= AVAILABLE SOCIAL MEDIA LINKS =

* Facebook
* Twitter
* Instagram
* WhatsApp
* Google+
* Pinterest
* Youtube
* TikTok
* Yelp
* Snapchat
* Discord
* Wordpress.com
* LinkedIn
* Tumblr
* Flickr
* MySpace
* Custom Link 1
* Custom Link 2

You can use the social media link as php code within your template files or as shortcode within the editor and your existing widgets.

== Support ==

Please submit your support requests through the WordPress forums for [Simple Theme Options](http://wordpress.org/support/plugin/simple-theme-options).

For any additional requests or suggestions, please contact us directly at hello@chrsinteractive.com

== Installation ==

1. Upload the folder 'simple-theme-options' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Within your dashboard navigation click on 'Simple Tracking'
